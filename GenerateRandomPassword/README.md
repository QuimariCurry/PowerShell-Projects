----------------------------------------------
GenerateRandomPassword
----------------------------------------------
A simple PowerShell script that generates secure, randomized passwords. This project helped me practice PowerShell fundamentals while building an automation tool for system administration and cybersecurity tasks.

----------------------------------------------
Features
----------------------------------------------
* Customizable password length
* Use of strong mix of characters
* Beginner- friendly script
* Easy to modify or expand with more options

----------------------------------------------
How to Run
----------------------------------------------
* Open PowerShell and run ".\GenerateRandomPassword.ps1"
* To specify a custom password length run ".\GenerateRandomPassword.ps1 -Length <number>
