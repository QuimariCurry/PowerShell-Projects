----------------------------------------------
SystemInformationTool
----------------------------------------------
A PowerShell script that collects your system information like OS details, hardware space, and network configuration. This project helped me build on my automation skills for system administration and cybersecurity tasks.

----------------------------------------------
Features
----------------------------------------------
* Interactive menu with 3 options:
    1. View System Info
    2. Save Report to Desktop
    3. Exit
* Collects OS, hardware, and network details
* Beginner-friendly script structure
* Easy to expand for more system checks

----------------------------------------------
How to Run
----------------------------------------------
* Open PowerShell and run ".\Get-SystemInfo.ps1"
* Choose one of the 3 menu options when prompted
