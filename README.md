# PowerShell-Projects
A collection of beginner-friendly PowerShell scripts that I have built in order to better understand automation system administration, and scripting fundamentals. Each project is organized in its own folder with a dedicated README and script file. 

## Purpose
This repo showcases practical PowerShell scripts that demonstrates scripting fundamentals, automation workflows, and Windows administration concepts. It is relevant to Windows administration and foundational scripting skills that are used in cybersecurity and SOC environments.

## How to Use
Navigate into any of the project folders to view its README and run the associated script. All scripts are written for Windows PowerShell and require no external modules.
