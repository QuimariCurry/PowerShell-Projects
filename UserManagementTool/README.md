----------------------------------------------
UserAccountManagementTool
----------------------------------------------
A beginner-friendly PowerShell script that automates basic local user account tasks like creating, disabling, listing, and resetting passwords. This project helped me practice PowerShell fundamentals.

----------------------------------------------
Features
----------------------------------------------
* Interactive menu with 4 options:
    1. Create a New User
    2. Disable a User
    3. Reset a User Password
    4. List All Users
* Uses built-in Windows account management cmdlets
* Clean script structure for beginners
* Helpful console messages for successful actions and errors

----------------------------------------------
How to Run
----------------------------------------------
* Open PowerShell and run ".\UserAccountManagementTool.ps1"
* Choose one of the menu options (1-4) when prompted
* Follow the script's guide through any required inputs (i.e. entering a username or password)
* After the selected action is completed, return to the menu for additional tasks or choose option 5 to exit the tool
